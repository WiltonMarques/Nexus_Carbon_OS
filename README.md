# 🛰️ Nexus Carbon-OS

**Infraestrutura Quantitativa e de Governança para Ativos Ambientais (CRAM)**

O Nexus Carbon-OS é um ecossistema *End-to-End* desenvolvido para a originação, auditoria, securitização e gestão de risco de ativos de carbono. O sistema integra inteligência geoespacial, auditoria semântica via LLM e modelos financeiros estocásticos para garantir a integridade total do lastro ambiental conforme o **PL 182/2024 (SBCE)**.

---

## 🚀 Arquitetura do Sistema

O projeto é dividido em 7 fases de execução autônoma:

1. **Sentinela Geoespacial:** Ingestão de dados de satélite e cálculo de biomassa (NDVI).
2. **Escudo RAG Anti-Greenwashing:** Auditoria semântica cruzando dados reais vs. narrativa corporativa.
3. **CRAM Securitization Engine:** Emissão de títulos e precificação paramétrica de carbono.
4. **Fortaleza Estocástica:** Simulação de Monte Carlo para detecção de Value at Risk (VaR).
5. **Hedge Structurer:** Precificação de derivativos via Black-Scholes para proteção de capital.
6. **Executive Terminal:** Dashboard interativo em Streamlit para visualização C-Level.
7. **Independent Audit Report:** Geração automática de laudos em conformidade com CVM 193 e CPC 46.

---

## 🛠️ Tecnologias Utilizadas

- **Core:** Python 3.10+
- **AI/LLM:** Google GenAI SDK (Gemini 2.5 Flash)
- **Finance:** SciPy (Black-Scholes), NumPy (Monte Carlo), yFinance (Market Data API)
- **Geo/Visual:** Folium, Plotly, Streamlit
- **Database:** PostgreSQL (Ledger de Transações)

---

## ⚖️ Conformidade e Normativos

O sistema foi desenhado para atender aos mais rigorosos padrões de governança:
- **CVM 193 / IFRS S2:** Divulgação de riscos financeiros relacionados ao clima.
- **CPC 46 / NBC TG 46:** Mensuração pelo Valor Justo (Marcação a Mercado).
- **CPC 01 / NBC TG 01:** Teste de Impairment automatizado via NDVI.
